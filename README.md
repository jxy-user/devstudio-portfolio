# DevStudio Portfolio

Full-stack developer portfolio & tools built with AI.

## 🚀 Projects

### Portfolio Landing Page
Modern Next.js + Tailwind CSS landing page with dark theme.

```bash
npm install
npm run dev      # Start dev server
npm run build    # Build for production → out/
```

### Figma Plugin: Batch Auto Layout
Apply auto-layout settings to multiple selected frames at once.

**To test:**
1. Open Figma desktop app
2. Plugins → Development → Import plugin from manifest
3. Select `figma-batch-autolayout/manifest.json`
4. Select multiple frames → Run plugin

## 🛠 Tech Stack

- **Frontend:** Next.js 16, React, TypeScript, Tailwind CSS
- **Plugins:** Figma Plugin API
- **Tools:** Git, VS Code, Codex AI

## 📦 Deploy

The landing page is pre-configured for static export (`out/` directory).
Deploy to Vercel or Netlify in one click.
